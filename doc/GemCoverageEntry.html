Overview

This class contains information describing which gemsets a gem is found in and what versions of the gem are installed in each gemset.

def flag_in_all_gemsets(gemsets)
  remaining =  gemsets - @gemsets_containing
  if (remaining.empty? or remaining == ['default'])  
    @in_all_gemsets = true
  end
end
