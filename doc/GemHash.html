GemHash

This class implements a Hash that uses a gem name as a key, and stores the GemCoverageEntry for that gem name. The GemCoverageEntry for a gem stores the gemsets a gem is found in and the versions of the gem found in those gemset.

Gems matching various conditions can be listed using the list_* methods.
